# Snake (React + Vite)

Minimal Snake game built with React and Vite. The code was generated by OpenAI Codex.

Quick start

1. Install dependencies

```bash
cd snake_game
npm install
```

2. Start dev server

```bash
npm run dev
```

Controls: Arrow keys or WASD to change direction. Space to Start/Pause. Click Restart to reset.

Notes: Requires Node.js (14+). This project uses Vite for development.
